faces-config.xml

Descrição:
Arquivo de configuração do JSF. 
Permite adicionar configurações adicionais se necessário.

Uso e Manutenção:

Configurações: Adicione configurações específicas do JSF conforme necessário. 
Geralmente não requer modificações frequentes.

web.xml

Descrição:
Arquivo de configuração do servlet e filtros da aplicação. 
Define a configuração do Faces Servlet, filtros de codificação e arquivos de boas-vindas.

Uso e Manutenção:

Configurações do Servlet: Define o servlet do JSF e suas configurações.
Filtros: Configura filtros como o de codificação de caracteres.
Arquivos de Boas-vindas: Define o arquivo de boas-vindas inicial da aplicação. 
Atualize conforme necessário para definir a página inicial ou adicionar novos filtros.
