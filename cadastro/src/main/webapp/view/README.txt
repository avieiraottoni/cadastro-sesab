Arquivos XHTML

pesquisaUsuario.xhtml

Descrição:
Tela de pesquisa de usuários com filtros por nome, CPF e período de cadastro. 
Permite ao usuário realizar buscas e visualizar os resultados.

Uso e Manutenção:

Filtros: Nome, CPF, Data de Início e Data de Fim. Atualize os campos e botões conforme 
necessário para suportar novos filtros ou funcionalidades.
Botões: Inclui opções para pesquisar e adicionar novos usuários.

cadastraUsuario.xhtml

Descrição:
Tela de cadastro de usuários, com campos para inserir ou editar informações como nome, 
CPF, e-mail, data de cadastro, perfil e endereços.

Uso e Manutenção:

Campos de Entrada: Nome, CPF, E-mail, Data de Cadastro. 
Atualize os campos e botões conforme necessário para suportar novas funcionalidades ou 
alterações no formulário de cadastro.
